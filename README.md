# REX Q1 2026 Report

Interactive dashboard for REX (Regional Exchange) tracing programme Q1 2026 performance.

**Live:** [https://mallyma1.github.io/rex-dashboard/](https://mallyma1.github.io/rex-dashboard/)

## What this covers

- Q1 trace output (520 traces across 8 freelancers)
- Month-on-month progress (Jan, Feb, Mar)
- TVS (Trace Value Score) and WIS (Withdrawal Impact Score) with full component breakdowns
- Cost efficiency trends
- Per-tracer performance with C1/C2 decomposition
- Operational improvements: NowPayments batch deposits, QC-gated payment model, sheet automations
- Q2 outlook

## Data sources

- Freelancer Google Sheets (Transactions + Already Traced tabs)
- Monthly freelancer tracker CSVs (Jan-Mar 2026)
- Retool entity export (3 Apr 2026)
